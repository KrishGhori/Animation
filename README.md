# Animation Box Project

##  Project Overview

This project demonstrates different **HTML, CSS, and JavaScript animations** using a moving ball inside a container.
The animation includes interactive features like starting/stopping movement, random motion, creating new balls, and gravity-based bouncing.

---


##  Features

* Start and Stop animation
* Ball trail / glow effect
* Random ball movement
* Click to create new balls
* Gravity bounce animation

---

##  Technologies Used

* HTML5
* CSS3 (Animations & Keyframes)
* JavaScript (DOM Manipulation & Events)

---


##  How It Works

1. A ball moves inside a container using CSS animation.
2. Users can start or stop the animation with buttons.
3. Random movement is generated using JavaScript.
4. Clicking inside the container creates new balls.
5. Gravity simulation makes the ball bounce naturally.
